# Final Practice

The purpose of these practices is to help **Front End** developers demostrate **AngularJS** and **Sass** concepts.

### Install

```sh
$ brew install nvm
$ npm install -g bower
$ npm install gulp-cli -g
```

### Start App

To run the examples follow the commands below:
```
$ npm install
$ bower install
$ cd Final-Practice/
$ ./src gulp serve
```
